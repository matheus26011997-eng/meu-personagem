# 🎮 RPG — Realm of Code

Mini jogo RPG com batalha em turnos feito com HTML, CSS e JavaScript puro.

## ✨ Funcionalidades

- Tela de login com escolha de personagem
- 3 classes jogáveis: Mago, Guerreiro e Arqueiro
- Sistema de batalha em turnos com inimigos aleatórios
- Habilidades únicas por classe com custo de mana
- Sons gerados em tempo real
- Ataques críticos e animações de dano

## 🛠️ Tecnologias

- HTML5
- CSS3 (animações e gradientes)
- JavaScript puro (sem frameworks)

## 🚀 Como jogar

1. Abra o arquivo `index.html` no navegador
2. Digite o nome do seu herói
3. Escolha sua classe
4. Entre no reino e batalhe!
